## Exercícios em C++ - Beecrowd
Este repositório foi criado para organizar listas de exercícios em C++ resolvidos no site Beecrowd (antigo URI Online Judge), a pedido do professor da disciplina de Programação Orientada a Objetos (POO). 

## Descrição
Aqui, você encontrará soluções em C++ para problemas do Beecrowd. Cada exercício é implementado de forma clara, visando praticar conceitos fundamentais como estruturas de controle, funções, cálculos matemáticos e lógica.

Linguagem: C++
Plataforma: Beecrowd
Objetivo: Prática para a aula de POO

-- Cada arquivo inclui:

Comentários explicando o código e o problema.
Código fonte completo e testável.
Referência ao ID do problema no Beecrowd.

## Contribuições
Este é um repositório pessoal para estudos. Se você encontrar erros ou sugestões de melhoria, sinta-se à vontade para abrir uma issue ou pull request. Lembre-se de que as soluções são baseadas nas minhas interpretações e podem não ser as únicas corretas.
